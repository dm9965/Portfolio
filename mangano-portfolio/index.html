Projects

TigerTalk GenCyber Social Media Hacking Activity

Social Media Password Hacking activity built for GenCyber Camp 2023 camp held at RIT. Campers are able to log in/create a user, post and view statuses, then are tasked with reverse engineering and hacking pre-existing user accounts based on the contents of their posts. This activity helps teach campers the importance of strong passwords, limiting personal data released to the public, and Cybersecurity fundamentals to protect themselves in the future

  • Developed using an Angular Typescript frontend and a Java RestAPI backend and Bootstrap CSS
  • Deployed and utilized by campers during 2023 GenCyber Camp at RIT
